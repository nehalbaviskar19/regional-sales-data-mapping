# Regional Sales & Data Mapping Dashboard 🗺️

## About this Project
This project analyzes Superstore Sales data and visualizes 
regional sales performance across United States using Power BI.

## Tools Used
- Power BI
- Power Query Editor (Data Cleaning)

## Dashboard Features
- 💰 KPI Cards (Total Sales, Profit, Discount)
- 🗺️ Map - Sales by State
- 📈 Line Chart - Sales Trend by Year
- 🍩 Donut Chart - Sales by Category
- 📊 Bar Chart - Profit by Sub-Category
- 🏆 Top 5 Customers by Sales
- 🔽 Interactive Filters (Region & Year)

## Dataset
Superstore Sales Dataset
